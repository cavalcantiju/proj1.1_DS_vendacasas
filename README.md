# proj1.1_DS_vendacasas
Projeto de Regressão Linear (OLS) para prever o preço de imóveis com base na área do primeiro andar. O modelo estimado foi: preço = 152900 + 6793.64·área. A variável é estatisticamente significativa e o R² = 0,377 indica poder explicativo moderado, sugerindo necessidade de mais variáveis.
